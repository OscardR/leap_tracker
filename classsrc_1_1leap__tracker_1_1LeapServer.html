Detailed Description

Main class to get data from the LEAP Motion controller.

It extends the Leap.Listener class and implements all the event methods defined in it. For more info, check the LEAP Motion API:

https://developer.leapmotion.com/documentation/skeletal/python/index.html

Starts transmission of tracking data.

Starts sending the current tracking values via ROS topics 'leap_tracker/joint_state_out', 'leap_tracker/pose_stamped_out' and 'leap_tracker/twist_stamped_out' to whichever LEAP tracking conversion services listening to these topics.
